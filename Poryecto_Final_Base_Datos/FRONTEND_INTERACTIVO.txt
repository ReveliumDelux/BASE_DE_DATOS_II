📋 ¿QUÉ PUEDES HACER AHORA?

✅ AGREGAR PACIENTES
   Dashboard → Pacientes → "Registrar Nuevo Paciente"
   • Cédula
   • Nombre completo
   • Edad
   • Sexo
   • Teléfono (opcional)
   • Alergias
   • Medicamentos
   ✓ SE GUARDA EN LOCALSTORAGE

✅ EDITAR PACIENTES
   Dashboard → Pacientes → "Editar"
   • Cambiar nombre
   • Cambiar edad
   ✓ CAMBIOS SE GUARDAN AUTOMÁTICAMENTE

✅ BORRAR PACIENTES
   Dashboard → Pacientes → "Borrar"
   • Elimina el paciente
   • Elimina de la cola también
   • Pide confirmación
   ✓ DESAPARECE DE TODAS LAS LISTAS

✅ HACER TRIAJE (IA ASISTIDO)
   Dashboard → Triaje
   1. Busca paciente (por cédula o nombre)
   2. Ingresa signos vitales:
      - Tensión Arterial (mmHg)
      - Frecuencia Cardíaca (bpm)
      - Frecuencia Respiratoria (rpm)
      - Saturación O2 (%)
      - Temperatura (°C)
   3. Describe síntomas
   4. Haz clic en "Analizar con IA"
   5. La IA sugiere prioridad automáticamente
   6. Validas manualmente
   7. Escribes justificación
   8. Confirmas triaje
   ✓ PACIENTE INGRESA A COLA AUTOMÁTICAMENTE

✅ VER COLA DE URGENCIAS
   Dashboard → Cola de Urgencias
   • Pacientes ordenados por prioridad
   • Visualización por color (Rojo > Naranja > Amarillo > Verde > Azul)
   • Filtro por prioridad
   • Tiempo de espera
   • Botón "Completar" para terminar atención
   ✓ SE ACTUALIZA EN TIEMPO REAL

✅ VER EXPEDIENTES
   Dashboard → Expedientes
   • Busca paciente
   • Ve todos sus datos
   • Ver triajes anteriores
   • Timeline de eventos
   • Historial médico
   ✓ INFORMACIÓN COMPLETA DISPONIBLE

✅ VER DASHBOARD
   • Métricas en tiempo real
   • Pacientes en cola
   • Pacientes por prioridad
   • Alertas críticas
   • Timeline de eventos
   ✓ TODO SE ACTUALIZA AUTOMÁTICAMENTE

✅ VER REPORTES
   Dashboard → Reportes
   • Estadísticas del día
   • Gráfica de distribución de triajes
   • Atenciones por hora
   • Indicadores clave
   ✓ DATOS EN VIVO

🔄 FLUJO COMPLETO EJEMPLO:

ESCENARIO: Paciente nueva llega a urgencias con síntomas respiratorios

1. AGREGAR PACIENTE (si es nueva)
   - Pacientes → Registrar Nuevo Paciente
   - Ingresa: Cédula 1111111111, Nombre "Rosa Martínez", 45 años
   - Click Registrar
   ✓ Rosa aparece en la tabla de pacientes

2. HACER TRIAJE
   - Triaje → Búsqueda: escribe "Rosa"
   - Signos vitales:
     TA: 130/85, FC: 110, FR: 22, SpO2: 94, Temp: 37.8
   - Síntomas: "Dificultad respiratoria, tos seca"
   - Click "Analizar con IA"
   ✓ IA sugiere "NARANJA - Muy Urgente" (92% confianza)

3. VALIDAR TRIAJE
   - Ves sugerencia: NARANJA
   - Modificas si es necesario (aceptas recomendación)
   - Escribes justificación: "Síntomas respiratorios con SpO2 en límite"
   - Click "Confirmar Triaje"
   ✓ Rosa ingresa automáticamente a cola en posición NARANJA

4. VER EN COLA
   - Cola de Urgencias
   - Rosa aparece en posición 2 (naranja)
   - Tiempo de espera: 0 min
   - Filtro solo naranja: solo ella aparece
   ✓ En tiempo real

5. CONSULTAR EXPEDIENTE
   - Expedientes → Busca "Rosa"
   - Ves todos sus datos
   - Triaje registrado: NARANJA
   - Justificación guardada
   ✓ Información persistente

6. COMPLETAR ATENCIÓN
   - Cola de Urgencias
   - Haz clic en "Completar" para Rosa
   - Se elimina de la cola
   ✓ Automático

❓ PREGUNTAS COMUNES:

P: ¿Dónde se guardan los datos?
R: En LocalStorage de tu navegador. No se envían a ningún servidor.

P: ¿Si cierro el navegador se pierden los datos?
R: NO. Los datos persisten en LocalStorage.

P: ¿Puedo usar esto en móvil?
R: Sí, es responsive (aunque optimizado para desktop).

P: ¿Necesito Internet?
R: NO. Funciona completamente offline.

P: ¿Necesito instalar algo?
R: NO. Solo abre el archivo HTML en tu navegador.

P: ¿Puedo tener múltiples usuarios?
R: En el mismo navegador se comparten datos. Para usuarios separados 
   necesitarías la API completa (próximo paso).

P: ¿Es seguro?
R: Los datos están en tu navegador local. Consideraciones HIPAA básicas 
   incluidas.
